투자 관련 메모/운영 히스토리는 이 경로를 기준으로 관리합니다.

## Canonical (Investment)
- `invest/docs/memory/README.md` (이 파일)
- `invest/docs/memory/invest_decisions.md` (투자 의사결정 로그)
- `invest/docs/memory/invest_todos.md` (투자 TODO/후속 작업)

## Rule
- 투자 관련 요약/결정/TODO는 `invest/docs/memory/*`에 기록.
- 상단 `memory/`는 개인/운영 컨텍스트 메모 중심으로 유지.
- 상단 메모에서 투자 맥락이 필요하면 본 경로를 먼저 참조.
