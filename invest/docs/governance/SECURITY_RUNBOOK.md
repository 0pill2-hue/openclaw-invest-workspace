## P0 즉시 대응 (적용 시작)
1. 비밀/세션 보호
   - `.env` 및 세션 파일 권한 600 유지
   - 키/세션 평문 노출 금지 (코드/로그)
2. 유출 대응
   - 키 유출 의심 시 즉시 키 폐기/재발급
   - 텔레그램 세션 의심 시 세션 종료 후 재인증
3. 수집기 격리
   - 수집기별 최소권한 원칙 적용 (계정/경로 분리)
4. 무결성
   - 원본 수집 데이터 해시 기록(SHA-256)
5. 로그
   - 토큰/세션/개인식별 정보 마스킹
