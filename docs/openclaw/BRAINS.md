주인님 기준 SSOT: 뇌는 2개만 사용한다.

## 1번 뇌 — GPT-5.4 (메인)
- 설계/판단/코딩/검증/최종 의사결정의 주력
- 중요 변경의 최종 리뷰/승인안 작성 담당

## 2번 뇌 — 로컬뇌 (보조)
- 폴백(장애 시)
- 요약/압축
- 크롤링/수집
- 배치/반복 작업
- 단순 지원

## 운영 원칙
- 기본은 1번 뇌가 직접 수행
- 장시간/반복/대량/크롤링/배치/비동기 작업에서만 예외적으로 2번 뇌 또는 위임 사용
- 최종 판단과 제출 책임은 항상 1번 뇌에 둔다
