## 0. 범위와 정의

### 0.1 범위
- 본 문서는 **Stage 공통 알고리즘 스펙**을 정의한다.
- Stage별 구현 상세(특정 스크립트, 데이터 포맷, 산출물 경로)는 stage docs에 둔다.
- 본 문서의 스펙은 “전략 로직의 의미”를 정의하며, 구현은 이를 따라야 한다.

### 0.2 용어
- Universe: 투자 대상 집합(종목/ETF/지수/자산군)
- Signal/Factor: 종목 점수 산출 입력
- Rank: 점수 기반 순위/스코어
- Portfolio: 실제 편입/가중치 결과
- Gate: 결과 승격/차단 규칙(DRAFT/VALIDATED/PRODUCTION)
