# PRIVATE LOCAL DOCS POLICY

역할: **로컬 전용 문서 정책**.

## 원칙
1. 일부 운영 보조 문서는 로컬 전용이며 Git에 포함되지 않는다.
2. 공개 repo 검토 및 문서 정합성 기준은 Git tracked 문서만 해당한다.
3. 로컬 전용 문서는 canonical source 로 취급하지 않는다.
4. 공개 문서에서는 로컬 전용 파일에 직접 링크하지 않는다.

## 공개 문서 작성 규칙
- `docs/index.html`과 `docs/operations/OPERATIONS_BOOK.md`에서는 로컬 전용 문서의 개별 파일명을 나열하지 않는다.
- 로컬 전용 문서가 필요할 때는 이 정책 문서만 참조한다.
- 공개 canonical 판단은 tracked 파일 기준으로만 수행한다.
